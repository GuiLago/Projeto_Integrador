# PROJETO INTEGRADOR - ROBÔ SUMÔ AUTÔNOMO

### Desenvolvimento de Sistemas - SENAI BA

Este é um projeto desenvolvido no SENAI Feira de Santana pera conclusão das matérias de Fundamentos de Eletroeletrônica Aplicada e Lógica de Programação. Os conhecimentos adquiridos ao longo do semestre de 2025.2 foram posto em prática pelos integrantes da equipe meste desafio que envolve desde a montagem e idealização à programação do equipamento.

O projeto consiste em projetar, montar e programar um robô autônomo capaz de batalhar numa arena circular onde o principal objetivo é empurrar o robô adversário para fora da arena em uma partida de 3 rounds.

Para este projeto utilizamos a lógica na linguagem C/C++ com Arduino IDE para programação do robô autônomo. Em robótica utilizamos as placas de desenvolvimento Arduino (R3/Mega 2650), ponte H, sensores ultrassônicos, sensores infravermelho, motores DC e protoboard para ligação dos elementos.
